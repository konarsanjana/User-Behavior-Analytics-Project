# 👨🏻‍💻 Customer Behavior Analysis Project

This project demonstrates an end-to-end data analytics workflow using SQL, Python, and Power BI. It was developed as part of guided learning by following a tutorial from Amlan Mohanty, with a focus on gaining hands-on experience in real-world data analysis tasks.

## 📌 Project Overview
The goal of this project is to analyze customer purchasing behavior and generate insights that can support business decision-making.

### Key Steps:
- Data cleaning and exploratory data analysis using Python  
- Writing SQL queries to analyze customer patterns  
- Building an interactive dashboard in Power BI  
- Generating insights and basic business recommendations  

## 🎯 Learning Outcomes
- Gained practical experience working with real-world datasets  
- Improved skills in SQL, Python (Pandas), and Power BI  
- Understood how end-to-end data analytics projects are structured
